# Pannello Solare Termico Sottovuoto - Vitosol 300-TM

**Fonte:** Documentazione ufficiale Viessmann - Brochure 09/2017

## Panoramica

**Vitosol 300-TM** - Pannello solare termico sottovuoto ad alta efficienza della nuova generazione Viessmann. Tecnologia heat pipe con sistema brevettato **ThermProtect** per protezione automatica dal surriscaldamento. Ideale per case indipendenti e condomini.

Uno dei pannelli solari termici sottovuoto **più efficienti sul mercato** con tubi orientabili.

## Tecnologia Heat Pipe

```
VITOSOL 300-TM - Principio di funzionamento

Radiazione solare
       ↓
Tubo sottovuoto (isolamento termico eccellente)
       ↓
Assorbitore selettivo rame
       ↓
Fluido heat pipe (non è il fluido solare!)
       ↓
Evaporazione fluido → Trasferimento calore
       ↓
Scambiatore Duotec (tubo doppio)
       ↓
Fluido solare (acqua-glicole) → Boiler
       ↓
Acquisione acqua calda + Protezione antigelo
```

**Vantaggio Heat Pipe**: Il fluido solare NON scorre nei tubi - ciò garantisce:
- ✅ Isolamento termico superiore (sottovuoto)
- ✅ Rendimento anche con orientamento non ottimale (+/- 25°)
- ✅ Protezione antigelo permanente
- ✅ Manutenzione tubi senza svuotare impianto

## Funzionamento Estivo (Ottimale)

```
Estate - Radiazione solare 800+ W/m²
       │
       ├─ Tubo sottovuoto assorbe calore
       │
       ├─ Heat pipe evapora fluido
       │
       ├─ Scambiatore Duotec trasferisce calore
       │
       ├─ Fluido solare riscaldato 60-70°C
       │
       └─ Boiler pre-riscaldato
          └─ Pompa di calore (Vitocal) riduce carico
          └─ ACS quasi gratuita dal sole
```

**Protezione ThermProtect**: Se T_boiler = setpoint per ore:
- Sistema interrompe trasmissione calore
- Evita surriscaldamento boiler
- Protezione automatica senza manutenzione

## Protezione Invernale

In inverno il pannello è **automaticamente protetto**:
- Fluido heat pipe congela a -20°C (più basso dell'acqua)
- Non c'è rischio di ghiaccio nelle tubazioni
- Sistema sospeso se T < 5°C (non utile)
- Nessun intervento richiesto (completamente automatico)

## Installazione

### Posizionamento Flessibile
- ✅ Orientamento non perfetto: tubi regolabili +/- 25°
- ✅ Montaggio verticale (muri, facciate)
- ✅ Montaggio orizzontale (terrazze basse)
- ✅ Spazi ridotti (balconi, giardini stretti)

### Collegamento
```
Pannello Vitosol 300-TM
       ↓
Pompa solare (attiva se T_pannello > T_boiler + 5°C)
       ↓
Circuito solare autonomo (come SPCF 100L)
       ↓
Boiler (pre-riscaldamento ACS)
       ↓
Pompa di calore (integra solo delta T rimanente)
```

## Manutenzione

### Annuale
- ✅ Ispezione visiva vetro (pulizia se polvere/neve)
- ✅ Controllo pressione circuito (0,5-1,5 bar funzione)
- ✅ Verifica isolamento tubi (nessun danno)

### Ogni 5 anni
- 🔄 Verifica fluido heat pipe (colore, densità)
- 🔄 Controllo scambiatore (sedimenti calcari in zone dure)
- 🔄 Ispezione tubi sottovuoto (nessun offuscamento)

### Se Malfunzionamento
- ❌ NON tentare riparazioni interne
- ✅ Tubi danneggiati: sostituibili senza svuotare impianto
- ✅ Contattare tecnico Viessmann certificato

## Sistema ThermProtect (Brevettato)

```
Protezione automatica dal surriscaldamento:

IF T_boiler = T_setpoint (45-50°C ACS)
  AND Radiazione solare = ALTA (800+ W/m²)
  AND Prelievo acqua = ZERO per ore

THEN:
  ✓ Pompa solare → STOP
  ✓ Trasmissione calore → INTERROTTA
  ✓ Boiler protetto da surriscaldo
  ✓ Energia solare → NON dissipata

Quando T_boiler < T_setpoint:
  ✓ Pompa riattivata automaticamente
  ✓ Ritorno a funzionamento normale
```
