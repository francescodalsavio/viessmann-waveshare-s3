# Vitocal 100-S - Unità Esterna

**Fonte:** Datasheet tecnico ufficiale Viessmann - Rev. 06/2018

## Panoramica

**Unità esterna della Vitocal 100-S** - Componente esterno della pompa di calore aria-acqua reversibile. Posizionata in giardino, terrazza o parete esterna. Estrae calore dall'aria esterna (inverno) o scarica calore all'esterno (estate).

## Componenti Unità Esterna

```
UNITÀ ESTERNA VITOCAL 100-S
┌──────────────────────────────┐
│ 1. EVAPORATORE               │ ← Assorbe calore aria esterna
│    (scambiatore di calore)   │
│                              │
│ 2. VENTILATORE               │ ← Soffia aria sull'evaporatore
│    (assiale orientato)       │
│                              │
│ 3. COMPRESSORE               │ ← Pompa refrigerante (cuore)
│    (con inverter)            │    Modulazione continua potenza
│                              │
│ • Gas refrigerante R32       │    (basso GWP, ecologico)
└──────────────────────────────┘
```

## Funzionamento

### Riscaldamento Invernale (A7/W35)
```
Aria esterna (-5°C a +7°C)
       ↓
Ventilatore soffia aria → Evaporatore
       ↓
Calore trasferito a refrigerante (pressione bassa)
       ↓
Compressore pressurizza il gas
       ↓
Tubo gas refrigerante → Unità interna
       ↓
COP 4,37-4,72 = 4,5 kW calore per 1 kW elettrico
```

### Raffrescamento Estivo (A35/W18)
```
Ciclo inverso: acqua calda da unità interna
       ↓
Compressore (ciclo freddo)
       ↓
Evaporatore unità esterna scarica calore
       ↓
Ventilatore soffia aria calda all'esterno
       ↓
EER 3,61-4,20 = 3,9 kW raffreddamento per 1 kW elettrico
```

## Specifiche Tecniche

| Parametro | Valore |
|-----------|--------|
| **Refrigerante** | R32 (GWP 675, basso impatto) |
| **Evaporatore** | Scambiatore aria/gas |
| **Ventilatore** | Assiale ad alta efficienza |
| **Compressore** | Inverter (modulazione continua) |
| **Range aria esterna** | -5°C a +35°C |
| **Rumore** | ~42-50 dB(A) in funzione |
| **Collegamento** | Tubo gas refrigerante (non acqua) |
| **Distanza massima** | Fino a 30 metri dall'unità interna |
| **Posizionamento** | Terra, parete, terrazza (orientamento flessibile) |

## Protezione

✅ Protezione antigelo (ciclo invertito in standby)
✅ Protezione compressore (timing 5 min tra cicli)
✅ Protezione pressione (valvole limitatrici)
✅ Ventola di sicurezza (arresto automatico)

## Manutenzione

### Stagionale (Primavera/Autunno)
- ✅ Pulire griglia evaporatore (foglie, polvere)
- ✅ Verificare drenaggio condensa
- ✅ Controllare fissaggi vibratoria

### Annuale
- 🔄 Ispezione visiva completa
- 🔄 Verifica isolamento tubi gas
- 🔄 Test rumore/vibrazioni

### Se Malfunzionamento
- ❌ NON tentare riparazioni
- ❌ Contattare tecnico Viessmann certificato
- ❌ Gas refrigerante: solo personale autorizzato

## Protezione Invernale

Se temperature < -5°C prolungate:
- Sistema passa a modalità protezione
- Compressore sospende ciclo riscaldamento
- Resistenza elettrica ausiliaria attiva (se disponibile)
- Ventola riduce giri (protezione ghiaccio evaporatore)
