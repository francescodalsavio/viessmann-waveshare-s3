# Scaldacqua a Pompa di Calore - Vitocal 060-A

**Fonte:** Scheda tecnica ufficiale Viessmann

## Specifiche Tecniche

| Parametro | Valore |
|-----------|--------|
| Capacità bollitore | 178 - 254 litri |
| Potenza utile | 1.3 kW |
| Temperatura max ACS | 62°C (sola pompa di calore) |
| COP (EN 255) | 3.1 @ aria 15°C / acqua 55°C |
| Classe energetica | A+ (scala G-A+++) |
| Fluido frigorifero | R1234-ze |
| Range temperatura aria | -5°C a +35°C |
| Installazione | Interna (cabina armadio/disimpegno) |

## Funzionamento

- **Aria ricircolata**: Assorbe calore da aria ambiente, genera ACS 62°C
- **Aria canalizzata**: Assorbe da esterno via adattatore fornito
- **Efficienza**: Riduce umidità, migliora qualità aria
